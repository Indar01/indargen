# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | ✅ Active Support  |
| 1.0.x   | ⚠️ Limited Support |

## Reporting a Vulnerability

If you discover a security vulnerability in IndarGen, please:

1. **Do NOT** open a public GitHub issue
2. Email the details to: security@indargen.dev
3. Include a detailed description and steps to reproduce
4. Allow 48 hours for initial response

We take all reports seriously and will respond promptly.

## Responsible Disclosure

We follow coordinated disclosure:
- Report received → acknowledged within 48 hours
- Investigation → patch development
- Public disclosure → after patch release (typically 7-14 days)
